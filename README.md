# Arsitektur-Data-Komponen-Data-Analytics
Repository ini berisi implementasi simulasi arsitektur data analytics modern menggunakan Python. Proyek mencakup proses ingestion, transformation, dan penyimpanan data dalam struktur data lake sederhana (raw, clean, curated). Dataset Superstore digunakan sebagai studi kasus untuk memahami alur data dari sumber mentah hingga siap analitik.
